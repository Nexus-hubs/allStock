# allStock - Financial Intelligence Platform

A minimal, elegant financial intelligence website with a ChatGPT-style interface featuring black background and beige lettering.

## Features

- **Real-time Stock Data**: Get live stock prices, market cap, volume, and other key metrics
- **Currency Exchange Rates**: View current exchange rates for major currencies
- **Tech News Integration**: Stay updated with the latest tech and stock news
- **Minimal UI**: Clean, modern interface inspired by OpenAI ChatGPT design
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Technologies Used

- Pure HTML5, CSS3, and JavaScript (no frameworks required)
- Yahoo Finance API for stock data
- Exchange Rate API for currency data
- Modern CSS Grid and Flexbox layouts

## How to Use

1. **Open the Website**: Simply open `index.html` in your web browser
2. **Search for Stocks**: Enter a stock symbol (e.g., AAPL, GOOGL, MSFT) in the search bar
3. **Quick Actions**: Click on the quick action buttons for popular tech stocks
4. **View Data**: Explore real-time stock prices, currency rates, and related news

## Supported Stock Symbols

- **AAPL** - Apple Inc.
- **GOOGL** - Alphabet Inc. (Google)
- **MSFT** - Microsoft Corporation
- **TSLA** - Tesla, Inc.
- **NVDA** - NVIDIA Corporation
- And many more...

## Design Philosophy

The website follows a minimal, elegant design approach:
- **Black Background** (#0d0d0d): Easy on the eyes, modern aesthetic
- **Beige Text** (#ece8dd): Soft, readable typography
- **Clean Layout**: Focus on content, minimal distractions
- **Smooth Interactions**: Subtle animations and transitions

## API Configuration

The website uses the Yahoo Finance API as configured in `yahoo_stock_api.json`:
- Endpoint: `https://query1.finance.yahoo.com/v7/finance/quote`
- Real-time stock quotes with comprehensive market data
